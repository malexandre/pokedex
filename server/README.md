# Mediflash PokeApi

Mediflash PokeApi bridge created using [Nest](https://github.com/nestjs/nest).

## Installation

**You should use Node 18.16.0.**

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
